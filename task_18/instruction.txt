1. Сгенерируйте буквенные обозначения мест
Conda хочет сгенерировать буквенные обозначения мест в своих самолетах. Самолет состоит из рядов кресел. В каждом ряду 4 места. Места в каждом ряду всегда обозначаются как A, B, C и D. Первое место в ряду — A, второе — B и так далее. После D нумерация начинается заново с A.

Реализуйте функцию generate_seat_letters(<number>), которая принимает int и содержит количество букв, которые нужно сгенерировать. Функция должна возвращать итерируемый объект с буквами.

>>> letters = generate_seat_letters(4)
>>> next(letters)
"A"
>>> next(letters)
"B"


2. Создание мест
Conda хочет систему, которая может генерировать заданное количество мест для их самолетов. В каждом самолете есть 4 места в каждом ряду. Строки определяются с помощью чисел, начинающихся с 1 и идущих вверх. Места должны быть заказаны следующим образом: 1A, 1B, 1C1D, 2A 2B2C, 2D 3A3B, 3C3D,,,,,,,,,,,,,,,,,,,,,,,,,,, ...

Вот такой пример:

x	1	2
Ряд	5	21
Буква места	A	D
Результат	5А	21D
Многие авиакомпании не размещают на своих рейсах ряд под номером 13 из-за суеверий среди пассажиров. Авиакомпания Conda Airlines также придерживается этого правила, поэтому убедитесь, что вы не создаете места для ряда под номером 13.

Реализуйте функцию generate_seats(<number>), которая принимает int и содержит информацию о том, сколько мест нужно сгенерировать. Функция должна возвращать итерируемый объект с заданными местами.

>>> seats = generate_seats(10)
>>> next(seats)
"1A"
>>> next(seats)
"1B"


3. Назначьте места пассажирам
Теперь, когда у вас есть функция, которая генерирует места, вы можете использовать ее для назначения мест пассажирам.

Реализуйте функцию assign_seats(<passengers>), которая принимает list имен пассажиров. Функция должна возвращать словарь, в котором passenger является ключом, а seat_number — значением.

>>> passengers = ['Jerimiah', 'Eric', 'Bethany', 'Byte', 'SqueekyBoots', 'Bob']

>>> assign_seats(passengers)
{'Jerimiah': '1A', 'Eric': '1B', 'Bethany': '1C', 'Byte': '1D', 'SqueekyBoots': '2A', 'Bob': '2B'}


4. Коды билетов
Авиакомпания Conda Airlines хотела бы иметь уникальный код для каждого билета. Поскольку это крупная авиакомпания, у нее много рейсов. Это означает, что на одном рейсе может быть несколько мест с одинаковыми номерами. Авиакомпания хочет, чтобы вы создали систему, которая генерирует уникальный код билета длиной 12 символов для идентификации.

Этот код начинается с assigned_seat, за которым следует flight_id. Остальная часть кода дополняется 0s.

Реализуйте функцию generate_codes(<seat_numbers>, <flight_id>), которая принимает list из seat_numbers и string с номером рейса. Функция должна возвращать generator с результатом ticket_number.

>>> seat_numbers = ['1A', '17D']
>>> flight_id = 'CO1234'
>>> ticket_ids = generate_codes(seat_numbers, flight_id)

>>> next(ticket_ids)
'1ACO12340000'
>>> next(ticket_ids)
'17DCO1234000'
